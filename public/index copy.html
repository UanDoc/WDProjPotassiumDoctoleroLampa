Reflection about the CS web design project:
The Computer Science Web design project allowed us (Uan and Rey) to explore our creativity with each and every web page that we developed as a result of this project. Aside from creativity, we also learned how to use our knowledge in Mathematics to further develop the JavaScript portion of this project. To be honest, this project was sort of difficult for the both of us due to our lack of experience with coding in both Javascript and HTML but we were able to produce a website that we are proud of.
The CS web design let us use the different lessons we've learned in computer science in creative ways. We also learned how to use apply our lessons in math in order to do the jscode portion of the website. Overall the coding portion was a little difficult because we both didn't have a deep understanding of Javascript & HTML but in the end we we're both satisified at the end result.
